---
name: deep-review
description: Comprehensive code review through systematic analysis
usage: /deep-review
---

Perform a comprehensive, systematic review of the codebase to identify bugs, security vulnerabilities, and quality issues, then automatically fix all the problems. This command delegates to the code-review-expert agent for thorough analysis and automatic implementation of fixes.

## When to Use

Use for comprehensive codebase analysis including:

- Before major refactoring efforts
- After completing significant features
- When preparing for code reviews or audits
- To identify technical debt and improvement opportunities
- When performance issues are suspected
- Before important releases or milestones

The code-review-expert agent will handle the complete review workflow: analyzing the codebase for bugs and vulnerabilities, automatically implementing fixes.
