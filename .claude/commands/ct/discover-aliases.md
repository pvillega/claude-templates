# Discover Shell Aliases

Scan shell aliases to identify tool replacements and generate documentation for Claude Code.

## Your Task

### Step 1: Run `alias` Command

Execute:
```bash
alias 2>/dev/null
```

### Step 2: Identify Tool Replacements

Parse the alias output to find these known replacement patterns:

| Original | Look for these replacements |
|----------|----------------------------|
| cat | bat, batcat |
| ls | eza, exa, lsd |
| grep | rg, ripgrep |
| find | fd, fdfind |
| du | dust, duf |
| df | duf |
| top | htop, btm, bottom |
| sed | sd |
| diff | delta |
| vim | nvim |
| man | tldr |

For each alias line like `cat='bat'` or `ls='eza --icons'`, extract:
- The original command (left of `=`)
- The replacement tool (first word after `=`, strip quotes)

### Step 3: Generate `~/.claude/Aliases.md`

Write to `~/.claude/Aliases.md` with this structure:

```markdown
# Shell Tool Aliases

> Auto-generated by /ct:discover-aliases

## Detected Replacements

[For each detected replacement, add a section:]

### [original] -> [replacement]

`[original]` is aliased to `[full alias value]`

**Key differences:**
[Use the reference below for each tool]

---
```

### Reference: Key Differences by Tool

**cat -> bat:**
- Syntax highlighting + paging by default
- Use `-pp` for plain output (no colors, no paging)
- Use `command cat` to bypass alias

**ls -> eza/exa:**
- Icons/colors by default
- Use `-T` for tree view (not `-R`)
- Use `command ls` to bypass alias

**grep -> rg:**
- Respects .gitignore by default
- Use `-u` to search all files (ignore .gitignore)
- Use `-F` for literal string (no regex)
- Use `command grep` to bypass alias

**find -> fd:**
- Pattern is regex by default (use `-g` for glob)
- Respects .gitignore by default
- Syntax: `fd pattern` not `find . -name pattern`
- Use `command find` to bypass alias

**du -> dust/duf:**
- Visual output with bars/colors
- Not suitable for scripting
- Use `command du` to bypass alias

**df -> duf:**
- Visual colored output
- Use `command df` to bypass alias

**top -> htop/btm:**
- Interactive TUI, not for scripting
- Use `command top` to bypass alias

**sed -> sd:**
- Different regex syntax
- Simpler: `sd 'find' 'replace'` vs `sed 's/find/replace/g'`
- Use `command sed` to bypass alias

**diff -> delta:**
- Visual colored output with syntax highlighting
- Use `command diff` to bypass alias

**vim -> nvim:**
- Modern features, Lua config
- Use `command vim` or `/usr/bin/vim` to bypass alias

### Step 4: Add @Aliases to CLAUDE.md

1. Read `~/.claude/CLAUDE.md`
2. Check if `@Aliases` already exists
3. If not present, add `@Aliases` at the end of the file

### Step 5: Report Summary

Display:
```
Alias Discovery Complete
========================
Detected: [list of original->replacement pairs]
Output: ~/.claude/Aliases.md
[If added] Added @Aliases to ~/.claude/CLAUDE.md
[If exists] @Aliases already in ~/.claude/CLAUDE.md
```

## Notes

- If no tool-replacement aliases are found, create a minimal file noting this
- Safe to run multiple times - overwrites `~/.claude/Aliases.md`
- Only add `@Aliases` to CLAUDE.md if not already present
