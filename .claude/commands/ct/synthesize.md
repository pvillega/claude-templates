# Multi-Framework Synthesis

## When to Use
- Multiple analytical frameworks applicable to same problem
- Need comprehensive understanding from different angles
- Risk of missing insights by using single framework
- Complementary perspectives expected to reveal different aspects

## Process
1. Framework Application - Apply each framework's analysis sequentially
2. Connection Building - Identify how insights from one framework inform others
3. Complementary Mapping - Note what each framework reveals uniquely
4. Convergence Identification - Find themes where multiple frameworks align
5. Integration Synthesis - Create unified understanding richer than any single framework

## Quality Checks
- Did each framework contribute unique insights?
- Were connections between frameworks made explicit?
- Does synthesis honor all perspectives (not just dominant one)?
- Is understanding richer than single-framework analysis?
