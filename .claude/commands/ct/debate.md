# Debate Facilitation

## When to Use
- Multiple valid perspectives exist on a decision
- Assumptions need challenging
- Trade-offs require clarification
- Consensus through synthesis (not compromise) is needed

## Process
1. Frame the disagreement (what specifically is contested?)
2. Evidence-based challenges (cite frameworks/data)
3. Assumption testing (what must be true for each view?)
4. Trade-off clarification (what's gained/lost in each approach?)
5. Synthesis seeking (higher-order resolution or strategic choice acknowledgment)

## Anti-Patterns
- Compromise instead of synthesis
- Personal attacks instead of evidence-based challenge
- Avoiding disagreement instead of surfacing it
- Declaring "right answer" instead of exploring trade-offs
