# SuperClaude Commands Summary

This document provides a comprehensive overview of all SuperClaude workflow automation commands. Each command orchestrates specialized agents and tools to perform complex software development tasks with intelligent automation and best practices.

## Analysis & Diagnosis

### analyze.md
Comprehensive code analysis tool that evaluates projects across multiple domains (quality, security, performance, architecture). It performs multi-domain analysis with intelligent file discovery, severity-based prioritization of findings, and generates actionable recommendations with comprehensive reporting and metrics.

### explain.md
Provides clear explanations of code, concepts, and system behavior with educational clarity. Features multi-persona coordination (educator, architect, security), framework-specific explanations via Context7, and adaptive explanation depth based on audience complexity with progressive learning approaches.

### troubleshoot.md
Diagnoses and resolves issues in code, builds, deployments, and system behavior through systematic root cause analysis. Features multi-domain troubleshooting (bug, build, performance, deployment), structured debugging methodologies with hypothesis testing, and safe fix application with verification and documentation.

## Implementation & Improvement

### design.md
Creates comprehensive design specifications for system architecture, APIs, components, and databases. Provides requirements-driven design with industry best practices, generates multiple format outputs (diagrams, specifications, code), and validates designs against scalability and maintainability standards.

### implement.md
Implements features and code with intelligent persona activation (architect, frontend, backend, security, qa) and MCP integration (Context7, Sequential, Magic, Playwright). Provides context-based implementation with framework-specific best practices, security validation, and comprehensive testing integration.

### improve.md
Applies systematic improvements to code quality, performance, and maintainability through multi-persona coordination (architect, performance, quality, security). Features framework-specific optimization via Context7, safe refactoring with validation, and comprehensive improvement documentation with rollback capabilities.

### cleanup.md
Systematically cleans codebases by removing dead code, optimizing imports, and improving project structure. Features multi-persona coordination (architect, quality, security), framework-specific cleanup patterns via Context7 MCP, and safety-first approach with backup and rollback capabilities for comprehensive code maintenance.

## Testing

### test.md
Executes tests with coverage analysis and automated quality reporting. Auto-detects test frameworks, generates comprehensive coverage reports, activates Playwright MCP for e2e browser testing, and provides intelligent test failure analysis with continuous watch mode support for development.

## Documentation

### document.md
Generates focused documentation for components, functions, APIs, and features in multiple formats (inline, external, API reference, user guides). Analyzes code structure, applies language-specific documentation patterns, and integrates with existing documentation ecosystems while maintaining consistency.

### index.md
Generates comprehensive project documentation and knowledge bases with intelligent organization. Features multi-persona coordination (architect, scribe, quality), Sequential MCP for systematic analysis, and Context7 for framework-specific patterns with cross-referencing and automated maintenance capabilities.

## Project Management

### task.md
Executes complex tasks with intelligent workflow management and multi-agent coordination. Features hierarchical task breakdown (Epic → Story → Task), multi-persona coordination across technical domains, intelligent MCP routing (Sequential, Context7, Magic, Playwright, Morphllm, Serena), and cross-session task persistence.

### workflow.md
Generates structured implementation workflows from PRDs and feature requirements. Features multi-persona orchestration (architecture, frontend, backend, security, devops), advanced MCP coordination with intelligent routing, cross-session workflow management with dependency mapping, and comprehensive quality gates for validation.

### spec-panel.md
Multi-expert specification review system using renowned software engineering experts (Karl Wiegers, Gojko Adzic, Martin Fowler, etc.). Features three analysis modes (discussion, critique, socratic), multiple focus areas (requirements, architecture, testing, compliance), and iterative improvement processes with quality metrics and actionable recommendations.

## Git Operations

### git.md
Handles Git operations with intelligent automation including conventional commit message generation, branch management, and workflow optimization. Provides smart status analysis, guided merge conflict resolution, and consistent branch naming conventions with workflow recommendations.

## Session Management

### pm.md
Default orchestration agent that coordinates all sub-agents and manages workflows seamlessly (always active at session start). Features comprehensive PDCA cycle execution, Serena MCP memory integration for cross-session persistence, self-correcting execution with root cause analysis, and zero-token baseline with dynamic MCP tool loading.

### load.md
Manages session lifecycle with Serena MCP integration for project context loading. Handles session initialization, cross-session persistence, memory retrieval, and project activation with performance-critical operations (<500ms target) and comprehensive context validation.

### save.md
Manages session context persistence using Serena MCP for cross-session memory management. Handles automatic checkpoint creation, discovery archival, session summary generation, and ensures cross-session compatibility with comprehensive validation for seamless session continuation.

## Reference

### help.md
Displays comprehensive reference documentation for all available SuperClaude commands and framework flags. Lists commands in tabular format with descriptions, explains behavioral flags (mode activation, MCP servers, analysis depth, execution control), and provides usage examples without executing any operations.
